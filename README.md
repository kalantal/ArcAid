# ArcAid

ArcAid is an EarthDay Hack-A-Thon project by Justin, Komron, and Craig (...& Rebecca...). <br />

With Tinder-Like aesthetics, you can swipe for charity in pre-specified increments as you see fit! <br />

https://arcaid.cc <br />
